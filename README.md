Eklavya – Accessible Examination Ecosystem for Visually Impaired Students
Overview

Traditional exam evaluation for visually impaired students often relies on human scribes, which introduces bias, interpretation errors, and a lack of transparency. Additionally, specialized digital Braille hardware is expensive, limiting accessibility for many students.

Eklavya is a zero-cost, software-driven examination ecosystem that enables visually impaired students to take exams independently using a standard keyboard as a 6-dot Braille input interface.

The platform ensures fair, transparent, and verifiable evaluation by capturing the student’s direct digital input rather than relying on third-party scribes.

Key Features
Accessible Braille Input

Converts a standard keyboard into a 6-dot Braille interface

No expensive Braille hardware required

Enables independent exam writing

NLP-Based Autocorrection

Real-time Natural Language Processing (NLP) corrections

Improves accuracy of typed responses

Text-to-Speech Feedback

Instant audio feedback to guide visually impaired users

Helps confirm typed inputs and navigation

Teacher Portal Dashboard

Educators can:

Generate tests from a built-in question library

Monitor real-time student progress

Evaluate answers objectively

Transparent Evaluation

Direct digital capture of student responses

Eliminates scribe bias and interpretation errors

Audit Trail for Grievance Redressal

Maintains 100% verifiable digital records

Enables fair and transparent exam review processes

How It Works

Student types using Braille key combinations on a standard keyboard.

System converts Braille inputs into text in real time.

NLP autocorrect improves response accuracy.

Text-to-Speech confirms input to the student.

Responses are sent to the Teacher Portal.

Teachers can review, grade, and track progress.
